# 📦 公众号文章收集系统 - 完整技能清单

## 🎯 核心技能

### 1. 文章提取技能 ✅
**文件**: `extract.js`  
**功能**: 
- 提取微信公众号文章完整内容
- 提取标题、作者、正文、发布时间
- 提取所有图片URL和元信息
- 支持任意网页链接

**使用**:
```bash
node extract.js "https://mp.weixin.qq.com/s/xxxxx"
```

---

### 2. 智能分类技能 ✅
**文件**: `classifier.py`  
**功能**:
- 自动识别文章类型（技术、产品、运营、设计等10+分类）
- 基于关键词权重计算分类得分
- 自动生成5个相关标签
- 支持自定义分类规则

**分类列表**:
- 技术、产品、运营、设计
- 商业、职场、生活、金融
- 教育、科技资讯

---

### 3. 文档格式化技能 ✅
**文件**: `formatter.py`  
**功能**:
- 三种格式风格：标准、卡片、Newsletter
- 自动排版和分段
- 生成精美报告文档

---

### 4. 图片处理技能 ✅
**文件**: `image_processor.py`  
**功能**:
- 提取文章中所有图片URL
- 图片上传到飞书云空间
- 创建文档图片块
- 支持图片批量处理

---

### 5. 定时任务技能 ✅
**文件**: `scheduler.py`  
**功能**:
- 创建定时收集任务
- 支持每日/每周/每月重复
- 任务管理和调度
- 后台自动执行

**使用**:
```bash
# 创建定时任务
python3 scheduler.py create --type collect --time "09:00" --repeat daily

# 启动调度器
python3 scheduler.py run
```

---

### 6. 摘要生成技能 ✅
**文件**: `summarizer.py`  
**功能**:
- 自动生成200-500字摘要
- 统计文章来源和分类
- 提取精选文章
- 生成分类报告

---

### 7. 消息推送技能 ✅
**文件**: `notifier.py`  
**功能**:
- 发送飞书消息通知
- 支持邮箱推送
- 支持Webhook推送
- 发送每日报告

---

### 8. 报告生成技能 ✅
**文件**: `reporter.py`  
**功能**:
- 生成每日/每周报告
- 创建飞书文档报告
- 统计分析和图表
- 自动发送通知

---

## 🚀 综合技能

### 9. 完整文章收集技能 ✅（主技能）
**文件**: `collector_final.py`  
**功能**:
- 一键完成全流程
- 自动提取、分类、摘要
- 图片提取和存储
- 推送到飞书表格和文档
- **包含原文链接和摘要内容**

**使用**:
```bash
# 完整收集
python3 collector_final.py --url "文章链接" --mode both

# 仅表格
python3 collector_final.py --url "文章链接" --mode table

# 仅文档
python3 collector_final.py --url "文章链接" --mode doc
```

---

### 10. 表格图片展示技能 ✅
**文件**: `collector_table_image.py`  
**功能**:
- 图片URL存储到表格
- 封面图链接展示
- 图片列表完整记录
- 无需额外权限

## 🎓 使用建议

### 快速开始
```bash
# 1. 安装依赖
cd article-collector
npm install
pip3 install -r requirements.txt

# 2. 配置飞书凭证
cp .env.example .env
nano .env  # 填入飞书 App ID 和 Secret

# 3. 开始使用
python3 collector_final.py --url "文章链接"
```

### 最佳实践
1. 使用 `collector_final.py` 作为主程序
2. 文章会自动分类和生成摘要
3. 图片URL会自动提取和存储
4. 文档包含原文链接，方便回溯

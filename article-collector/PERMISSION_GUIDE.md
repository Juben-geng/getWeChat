# 飞书权限配置指南

## 需要的权限列表

### 已配置权限
- ✅ `bitable:app` - 多维表格操作

### 需要添加的权限

| 权限名称 | 权限代码 | 用途 |
|---------|---------|------|
| 查看、评论、编辑和管理文档 | `docx:document` | 创建带图片的飞书文档 |
| 查看、评论、编辑和管理云空间中文件 | `drive:drive` | 上传图片到飞书云空间 |

## 配置步骤

### 方式一：手动添加

1. 打开 [飞书开放平台](https://open.feishu.cn/)
2. 进入你的应用管理页面
3. 点击左侧菜单「权限管理」
4. 点击「申请权限」
5. 搜索并添加以下权限：
   - 搜索 "文档" → 添加「查看、评论、编辑和管理文档」
   - 搜索 "云空间" → 添加「查看、评论、编辑和管理云空间中文件」
6. 进入「版本管理与发布」
7. 创建新版本并提交审核

### 方式二：API 方式

使用以下 URL 快速添加权限（替换 `{APP_ID}` 为你的应用 ID）：

```
# 文档权限
https://open.feishu.cn/app/{APP_ID}/auth?q=docx:document,docx:document:create&op_from=openapi&token_type=tenant

# 云空间权限
https://open.feishu.cn/app/{APP_ID}/auth?q=drive:drive,drive:file:upload&op_from=openapi&token_type=tenant
```

## 权限说明

| 权限 | 功能 |
|------|------|
| `bitable:app` | 创建和管理多维表格 |
| `docx:document` | 创建带图片的飞书文档 |
| `drive:drive` | 上传图片到飞书云空间 |

## 临时方案（无需额外权限）

### 方案一：仅使用表格

```bash
python3 collector_final.py --url "文章链接" --mode table
```

### 方案二：图片URL存储到表格

图片URL已经存储在表格的字段中，可以：
1. 在飞书表格中查看图片URL
2. 点击URL在新标签页打开图片

### 方案三：生成Markdown文档

```bash
python3 formatter.py > article.md
```

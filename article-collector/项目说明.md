# 🎉 文章收集系统已创建成功！

## 📦 项目位置
`/workspace/article-collector/`

## 🚀 快速开始

### 第一步：配置飞书凭证

```bash
# 1. 进入项目目录
cd /workspace/article-collector

# 2. 复制环境变量模板
cp .env.example .env

# 3. 编辑配置文件（填入你的飞书凭证）
nano .env
```

需要填入：
- `FEISHU_APP_ID` - 飞书应用ID
- `FEISHU_APP_SECRET` - 飞书应用密钥

### 第二步：测试提取功能

```bash
# 测试提取微信文章（无需飞书凭证）
node extract.js "https://mp.weixin.qq.com/s/你的文章链接"
```

### 第三步：收集文章

```bash
# 收集单篇文章（同时创建文档和表格）
python3 collector.py --url "https://mp.weixin.qq.com/s/xxxx"

# 批量收集
python3 collector.py --file urls.txt

# 仅创建表格
python3 collector.py --url "https://mp.weixin.qq.com/s/xxxx" --mode table

# 仅创建文档
python3 collector.py --url "https://mp.weixin.qq.com/s/xxxx" --mode doc
```

## ✨ 核心功能

### 1. 文章提取
- ✅ 支持微信公众号文章
- ✅ 支持任意网页链接
- ✅ 自动提取标题、作者、正文、发布时间等
- ✅ 支持批量处理

### 2. 飞书表格汇总
自动创建多维表格，包含字段：
- 标题
- 作者
- 来源（公众号名称）
- 发布时间
- 原文链接
- 摘要
- 收集时间

### 3. 飞书文档汇总
自动创建格式化文档，包含：
- 文章标题
- 作者和来源信息
- 发布时间
- 原文链接
- 文章摘要
- 完整正文

## 🔑 获取飞书凭证

### 1. 创建飞书应用

1. 访问 https://open.feishu.cn/
2. 点击「开发者后台」→「创建企业自建应用」
3. 填写应用名称（如：文章收集助手）

### 2. 获取凭证

在应用详情页找到：
- App ID
- App Secret

### 3. 配置权限

在「权限管理」添加以下权限：

| 权限名称 | 权限代码 |
|---------|---------|
| 查看、评论、编辑和管理多维表格 | `bitable:app` |
| 查看、评论、编辑和管理文档 | `docx:document` |
| 查看、评论、编辑和管理云空间中文件 | `drive:drive` |

### 4. 发布应用

创建版本 → 提交审核 → 发布
